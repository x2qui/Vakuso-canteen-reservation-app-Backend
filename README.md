# Meal QR App

A full-stack app that allows users to reserve meals by scanning QR codes.

## 🛠️ Tech Stack
- Frontend: React Native
- Backend: Node.js & Express
- Database: MongoDB
- QR: QRCode API


## 🚀 Setup
1. Clone the repo
2. Run `npm install`
3. Add your `.env` file
4. Run with `npm start`

## ✅ Features
- QR Code generation and scanning
- Meal selection and reservations
- Authentication
